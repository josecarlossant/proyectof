MANUAL








Hola que tal en este pequeño instructivo te enseñaremos como crear un despliegue de una web site utilizando solamente sublime tex, GitHub,getbootstrap versión 5.3.x.




Comencemos:




1. Tendremos que hacer la programación en este caso utilizamos sublime text para cada aspecto deseado para que se muestre en la web site.








2. También nos apoyaremos en la páginagetbootstrap versión 5.3.x para poder agregarle un formato llamativo a nuestro web site.





3. Se crea una cuenta en GitHub page con un correo electrónico y una contraseña.





4. Una vez dentro de GitHub nos vamos a la parte superior izquierda New para crear una repositorio.





5. Mandará esta pantalla y rellenaremos los datos solicitados.





6. En la parte de abajo nos mostrara opciones las cuales hay que rellenar, en la parte de licencia, hay que anexarla, Apache Licencia 2.0 y darle en Créate Repository.





7. Mandará a la página donde ya fue creado el repositorio donde vamos a subir los archivos html para poder crear nuestra web site.





9. Una vez arriba los archivos le damos en Commit changes .





10. Pasamos a Settings seguido de: Pages .





11. Esperamos un momento para que nos genere el link de la página del repositorio que hemos creado y una vez generado, damos clic en el link

.



12.Mandará directo a nuestra página en nuestro buscador.





13. Mandará un link que esta página en la que estamos y podemos observar todo lo relacionado con el web site .
